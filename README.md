# Min Quiz Kamp

A simple multiplayer quiz game built with TCP. Players will play in pairs, and the total number of players is unlimited.


## Features & Usage

### Multiplayer - TCP:

![](https://github.com/ngocmai-do/MinQuizKamp/blob/master/documentation/multiplayer.png?raw=true)


### Number of rounds and the number of questions each round can be changed through a properties file:
Number of rounds of number of questions each around saved in a properties file (easy for updating)

![](https://github.com/ngocmai-do/MinQuizKamp/blob/master/documentation/properties%20file.png?raw=true)


### Choose quiz category & result update:

At the beginning of each round, players will take turns to choose a category. While one player is choosing, the other has to wait.

![](https://github.com/ngocmai-do/MinQuizKamp/blob/master/documentation/round%201%20and%202.png?raw=true)
![](https://github.com/ngocmai-do/MinQuizKamp/blob/master/documentation/round%203%20end%20game.png?raw=true)
